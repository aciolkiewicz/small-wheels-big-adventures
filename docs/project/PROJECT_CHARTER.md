# Małe Koła, Wielkie Przygody

## Project Charter

---

# 1. Purpose

**Małe Koła, Wielkie Przygody** is a content-first platform helping families with young children discover and plan memorable one-day trips.

The project combines carefully curated travel guides, cycling routes, family-friendly attractions and practical planning tools into a single ecosystem.

The long-term vision is to provide one trusted place where parents can easily answer a simple question:

> "Where should we go this weekend?"

---

# 2. Mission

Help parents spend more quality time with their children by making weekend planning effortless.

Instead of searching through dozens of websites, social media groups and blogs, families receive complete, ready-to-use trip plans.

---

# 3. Vision

Become the most trusted family trip planning platform in Poland.

The platform will eventually include:

- interactive website
- mobile application
- printable travel guides
- GPX routes
- offline maps
- seasonal recommendations
- community contributions

---

# 4. Target Audience

Primary audience:

- parents
- children aged 2–8 years
- active families
- cycling enthusiasts
- families looking for one-day trips

Secondary audience:

- grandparents
- tourists visiting central Poland
- preschool groups
- local tourism organizations

---

# 5. Problem Statement

Planning a family trip currently requires visiting multiple sources:

- Google Maps
- blogs
- Facebook groups
- weather forecasts
- restaurant reviews
- railway websites

Information is fragmented, outdated and rarely written from the perspective of parents with small children.

The project solves this problem by providing complete trip scenarios instead of isolated attractions.

---

# 6. Project Principles

## Family First

Every recommendation must prioritize children's needs.

---

## Quality over Quantity

Only exceptional trips become part of the guide.

Adding more routes is never more important than maintaining quality.

---

## Honest Recommendations

No hidden advertisements.

No sponsored rankings.

Every recommendation should be something we would confidently recommend to our own family.

---

## Practicality

Every route must answer practical questions:

- Where to park?
- Where to eat?
- Where is the nearest toilet?
- Where can children play?
- Is it suitable for a bike seat?
- Is there enough shade?

---

## Simplicity

Planning a trip should take minutes, not hours.

---

# 7. Scope

## Included

- cycling routes
- walking routes
- family attractions
- restaurants
- playgrounds
- picnic locations
- railway access
- GPX files
- printable guides
- mobile application
- website

---

## Excluded

The project is **not** intended to become:

- booking platform
- travel agency
- hotel comparison website
- restaurant review portal
- navigation system

Whenever possible, existing services will be integrated instead of duplicated.

---

# 8. Product Vision

The project is built around one central source of truth.

Content is created once and published everywhere.

Possible outputs include:

- website
- mobile application
- PDF guides
- newsletters
- social media
- printable maps

---

# 9. Success Metrics

Examples of future KPIs:

- published routes
- verified attractions
- active users
- returning visitors
- completed trips
- community contributions

Initial development focuses on quality rather than numerical goals.

---

# 10. Values

The project is guided by five values:

- Family
- Nature
- Simplicity
- Trust
- Quality

These values should influence every design and product decision.

---

# 11. Decision Making

When multiple solutions exist, the preferred option should:

1. improve user experience
2. simplify maintenance
3. maximize content reusability
4. minimize long-term technical debt

---

# 12. Long-Term Vision

The project begins as a regional travel guide focused on Łódź and surrounding areas.

Over time it may expand into:

- all Polish regions
- multilingual versions
- community-created routes
- educational content
- premium features
- partnerships with local organizations

Expansion should never compromise editorial quality.

---

# 13. Definition of Success

Success is not measured by the number of published routes.

Success is achieved when families spend more meaningful time together because planning trips has become easy, enjoyable and trustworthy.

If parents can confidently choose a destination in less than five minutes, the project fulfills its mission.
