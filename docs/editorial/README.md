# Editorial Handbook

## Purpose

This section defines the editorial standards for the **Małe Koła, Wielkie Przygody** project.

Its purpose is to ensure that every published route, recommendation and article follows the same level of quality, consistency and trustworthiness.

Editorial quality is one of the project's core values.

## Principle

Every family should receive the same high-quality experience regardless of which route they choose.

Editorial consistency is more important than publishing quickly.
