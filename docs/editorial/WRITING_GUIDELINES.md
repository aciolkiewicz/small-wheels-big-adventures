# Writing Guidelines

## Purpose

This document defines the writing style used across the **Małe Koła, Wielkie Przygody** ecosystem.

It ensures every article, route and recommendation feels like it was written by the same editorial team.

Writing should always support the project's mission:

> Helping families spend more time together and less time planning.

---

# Core Principle

We write for parents.

Not for search engines.

Not for social media.

Not for ourselves.

Every sentence should make planning easier.

---

# Writing Philosophy

We do not describe places.

We describe experiences.

Instead of listing attractions, we explain how families can enjoy them.

---

# Our Reader

When writing, imagine one specific reader.

- Parent
- Child aged 2–8
- Weekend trip
- Limited planning time
- Wants practical advice

If a sentence does not help this reader, remove it.

---

# Tone of Voice

Our tone is:

- calm
- friendly
- practical
- encouraging
- trustworthy

Never:

- sensational
- exaggerated
- sarcastic
- overly emotional

---

# Write Like a Local Friend

Imagine you have already visited the place.

You are recommending it to a close friend.

Example:

❌

"There is a large playground."

✅

"The playground is fenced and visible from the café terrace, so parents can enjoy a coffee while keeping an eye on their children."

---

# Practical Information First

Readers need answers before stories.

Every route should answer practical questions early:

- Where do we park?
- Is it suitable for a bike seat?
- Is there shade?
- Are toilets available?
- Can we buy food?
- How much time should we plan?

---

# Show, Don't Sell

Avoid marketing language.

Instead of:

> Amazing attraction!

Describe what makes it useful.

Example:

"There are several picnic tables under mature trees, making it a comfortable stop on hot days."

---

# Use Concrete Details

Prefer:

- distances
- estimated time
- surface type
- child-friendly facilities

Avoid vague descriptions.

---

# Be Honest

Every place has limitations.

Mention them.

Examples:

- difficult hill
- muddy path after rain
- no toilets
- limited parking
- crowded during weekends

Honest recommendations build trust.

---

# Structure Matters

Use short paragraphs.

Prefer descriptive headings.

Use bullet lists only when they improve readability.

Avoid long blocks of text.

---

# Language

Use simple vocabulary.

Avoid unnecessary technical or tourist jargon.

Write naturally.

---

# Storytelling

Every route should tell a small story.

Beginning:

Why visit?

Middle:

What happens during the trip?

Ending:

Why is it memorable?

---

# Family Perspective

Whenever possible describe experiences from a child's perspective.

Examples:

- feeding ducks
- climbing a small hill
- collecting leaves
- watching trains
- eating ice cream

These moments create memories.

---

# Recommendations

Recommend only places we would personally choose.

If we are unsure, we do not recommend them.

---

# SEO

SEO supports content.

Content never exists only for SEO.

Keywords should appear naturally.

Never sacrifice readability for search rankings.

---

# Inclusive Language

Write for every family.

Avoid assumptions.

Examples:

Instead of:

"Mom can prepare lunch."

Prefer:

"Parents can prepare lunch."

---

# Things We Never Write

Avoid phrases like:

- hidden gem
- must see
- breathtaking
- unforgettable
- best in Poland
- everyone should visit

These expressions are subjective and often overused.

---

# Preferred Expressions

Prefer practical descriptions such as:

- easy to reach
- suitable for young children
- shaded area
- quiet place
- family friendly
- accessible by train
- ideal for a picnic

---

# Before Publishing

Ask yourself:

- Does this help parents?
- Is every important fact verified?
- Is anything exaggerated?
- Would I recommend this to my own family?
- Would I trust this article if I found it online?

If any answer is "No", improve the content before publishing.

---

# Writing Checklist

Every article should be:

- useful
- honest
- practical
- easy to read
- inspiring
- factually correct
- family focused
